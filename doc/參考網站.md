## 2. OptiTrack「Build My System」(核心 UX 標竿) ★

- **網址**：<https://www.optitrack.com/systems?camera=FL3-BL-FSBP-3&numberOfCameras=6#animation>
- **定位**：**汽車配置器式**的產品配置器——這就是本專案想做的互動體驗
- 提供日期：2026/06/26（含 3 張截圖）

### 核心互動：選項變動 → 左側模擬示意圖即時變化 ★★★

> 使用者最在意的點：每次選「場地 / 場景 / 相機數」，左邊的 3D 模擬示意圖都會跟著變。
> 例如 Flex 3 / 6 台 → 小場域 4 支腳架環繞人物；Prime X 22 / 16 台 → 大型 Truss 桁架環場。

### 版面結構（兩欄 + 下方明細）

```
┌──────────────────────────────┬─────────────────────────┐
│  左：動態模擬示意圖            │  右：配置面板 (sticky)    │
│  （隨選項即時變化的 3D 場景）  │  Application 應用情境      │
│   - 相機/腳架/桁架/人物擺位    │  Camera Type 相機型號     │
│   - 下方一行：型號 + 一句話描述 │  Number of Cameras 數量   │
│                              │  ─────────────           │
│                              │  Estimated Price 預估價    │
│                              │  [Send Me a Quote][Add Cart]│
└──────────────────────────────┴─────────────────────────┘
┌──────────────────────────────────────────────────────────┐
│  Camera Details            │  Volume Details              │
│  (解析度/範圍/延遲)         │  (擷取體積/設置面積/人數)     │
├──────────────────────────────────────────────────────────┤
│  System Contents — BOM 卡片（icon + 品名，網格排列）       │
├──────────────────────────────────────────────────────────┤
│  Recommended PC Specs — 條列                              │
└──────────────────────────────────────────────────────────┘
```

### 配置面板選項（截圖實錄）

| 區塊 | 選項 | 對應本專案 |
|------|------|-----------|
| **Application** | Animation / Movement Sciences / Robotics / Virtual Production / Virtual Reality（chip 多選一） | → 應用情境（軍/警/消…） |
| **Camera Type** | Flex 3 / Flex 13 / PrimeˣX 13 / 13W / 22 / 41 / 120（含小圖示卡片） | → **定位方式**（AL/Opti/UT/SX/WT/Vive） |
| **Number of Cameras** | 6 / 8 / 12 / 16 / 20 / 24（依相機型號動態變動可選範圍） | → **人數 / 場域大小** |
| **Estimated Price** | 即時計算總價（如 $12,031 / $76,429.50） | → **成本/報價** |
| CTA | Send Me a Quote（藍）/ Add to Cart（黑） | → 洽詢報價 / 加入清單 |

### 下方明細區（截圖實錄）

- **Camera Details**：Resolution、Passive Range、Active Range、Latency
- **Volume Details**：Capture Volume、Setup Area、Actors/Subjects
- **System Contents**：BOM 卡片網格（如 Flex 3、Sync Cable、Calibration Square、OptiHub 2、Motive:Body、各式線材、Security Key…）→ 對應我們的 **BOM 明細**
- **Recommended PC Specs**：條列硬體需求

### 設計語言（截圖觀察）

- 極簡白底、大量留白、黑色標題粗體、藍色點綴連結
- 選項用**圓角卡片 / chip**，選中態藍框
- 配置面板淺灰背景、圓角、輕陰影，**sticky 跟隨捲動**
- 價格大字醒目；CTA 雙按鈕（主藍 + 次黑）
