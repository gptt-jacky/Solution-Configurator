# 官網解決方案架構

> 來源：BigxReality 官網 solutions 頁內文（使用者於 2026/06/26 提供完整文字）。
> 公司：**GLOBAL POWER TECHNOLOGIES CO., LTD.（全球動力科技）**，BigxReality 為其 AI XR 品牌。
> 用途：作為配置器最上層的**應用情境（Application）分類**，等同 OptiTrack 的 Application 選擇器。

---

## 三大解決方案

```
解決方案
├─ 警勤模擬 (Police)      ← 配置器主戰場（含武器/定位/場域）
├─ 軍事模擬 (Military)    ← 配置器主戰場（含武器/載具/定位/場域）
└─ 智慧城市 (Smart City)  ← 數位雙生，非武器型配置（可能不納入 MVP）
```

---

## 1. 警勤模擬｜智慧 XR 警勤模擬訓練系統

- **對象**：警察，1 至多人，射擊+警勤武器+多元情境+多人協同
- **訓練裝備**：**警棍、辣椒水罐、電擊槍、手槍**（+ 各單位客製化設備）
- **訓練模式（可作課程模板）**：
  | 模式 | 重點 |
  |------|------|
  | 基礎訓練 | 單一警具操作、身體協調（警棍/手槍） |
  | 進階訓練 | 實景射擊、應對隨機移動武裝嫌犯、自我防禦 |
  | 言語降級訓練 | 教官角色扮演嫌犯，言語溝通緩和衝突（武力升降級） |
  | 情境模擬訓練 | 預設狀況、大腦即時反應、武力升降級 |
  | 課程編輯器 | 指揮官/教官自訂情境與任務流程 |
  | 訓練後學習機制 | 自動生成數位學習履歷 + 生理監測 |
- **方案特色**：安全可控環境、即時數據分析（裝備使用/行動軌跡/心律體溫）、降低彈藥與人事成本

## 2. 軍事模擬｜AI XR 合成化戰場（全亞洲唯一，整合五大模組）

- **涵蓋**：指揮訓練、兵力佈屬、單兵訓練、團隊作戰、裝備操作、載具駕駛、多軍聯合
- **五大產品模組**：
  | 模組 | 內容 | 武器/裝備 |
  |------|------|----------|
  | ① 武器裝備操作訓練 | 步兵槍械 + 野戰防空射擊 | 機槍、步槍、狙擊、迫擊砲觀測、**刺針飛彈發射模擬器**、火箭彈發射程序 |
  | ② 駕駛訓練系統 | 多軸動感平台 + XR，全地形、多人協同 | **悍馬車、快艇** |
  | ③ 兵棋圖台系統 | 指揮官兵力部署，**MIL-2525B 兵棋符號**、3D 多維度 AI 兵棋圖台 | （軟體） |
  | ④ 限制空間戰術訓練 (CQB) | 特勤、城市/巷弄/建築/商城、近身戰鬥、多人聯網 | 槍械 |
  | ⑤ 訓練後數據分析 | 一鍵錄影回放、彈道/環境/生理數據、終生學習履歷 | （軟體） |
- **方案特色**：提升作戰技能、提高決策效率、降低訓練風險

## 3. 智慧城市｜數位雙生 (Digital Twin)

- **技術**：IoT + 大數據 + AI，物理世界 1:1 虛擬化，3D 視覺化分析
- **應用**：智慧建築管理、智慧交通分析、智慧消防、智慧空側駕駛、災害預警與應變、智慧港埠管理、**智慧運動（XR 射擊運動訓練系統）**
- **方案特色**：數據整合、模擬與預測、資源管理、視覺化分析
- ⚠️ 此類偏監控/數位雙生，非「選武器+定位」型配置 → **MVP 可能不納入**，待確認

## 對配置器的含義

- **頂層選擇器 = 應用情境**：警勤模擬 / 軍事模擬（/ 智慧城市待定）。
- 選了情境後，**收斂可選武器與課程模板**：
  - 警勤 → 警棍/辣椒水/電擊槍/手槍 + 5 種訓練模式
  - 軍事 → 五大模組（武器操作/載具駕駛/兵棋/CQB/數據分析）
- 這層與 `武器與訓練課程清單.md`、`定位系統比較.md` 串接，構成完整篩選流程：
  **情境 → 武器/裝備 → 人數/場域 → 定位方式（自動排除）→ BOM/價格 → 示意圖**。

## MVP 範圍（使用者確認 2026/06/26）

- ✅ **頂層情境只做兩個：警勤模擬 + 軍事模擬**；**智慧城市排除**（數位雙生，非選武器型）。
- ✅ **人數、場地大小**為明確的配置選項（兩者數值範圍差很大，需可選/可輸入），
  並作為定位方式自動排除的依據。
