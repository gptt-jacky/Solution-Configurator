## 本版改動摘要（相對 v7）

1. ⭐【本版重點】**場景示意圖必須真的隨選擇即時更新**（目前勾現場條件畫面都一樣＝沒接上，要修）。
2. 🆕 **右上角語言切換**：繁體中文（預設）/ English / 日本語，全站文字即時切換。
3. 🆕 **OptiTrack 落地支架規則**：
   - 大型場地 + 不可架上方桁架 → OptiTrack 改用**落地支架**（**較便宜**），仍可用。
   - **超大型場地**：OptiTrack 只能**上方桁架**（**落地支架不可用**）；若超大型又不可架桁架 → 排除 OptiTrack。
4. 🆕 **現場條件精簡**：移除「需快速佈置(1hr)」，保留「需移動/非固定場館」→ 共 6 項。
5. 💤 **價格先不動**（沿用 v7 模型，本版不調折扣/落地支架成本細節）。
6. 其餘 v7 規格沿用（武器數量+折扣、定位推薦+備選、WT限1–4人、藍UI+紅logo、左視覺凍結右選單捲動、價格頂部凍結且手機一行、武器轉盤、hover上浮、Apple圓角+SF Pro…）。

> 註：完整「現場條件→排除哪個方案」矩陣仍待技術單位校正；本版先把**圖示連動**與已知的 OptiTrack 落地支架規則做對。

# 1. ⭐ 本版最重要：場景示意圖必須真的隨選擇即時更新

**場景示意圖改用「使用者在對話視窗直接貼上的示意圖」**（非資料夾路徑），依**定位方案**對應切換：

| OptiTrack 落地支架（多支腳架環繞人物） | 大型場地、未架桁架時的 OptiTrack |
| OptiTrack 上方桁架 | 用上方桁架的 OptiTrack |
| Antilatency 地墊/燈條 | Antilatency |

| 免布置（無外部設備） | Ultimate / Wrist / Sensoryx 共用 |

> 使用者貼圖時會標明哪張對應哪個方案；請依此對應。若某狀態暫無圖，沿用既有 SVG 場景。

目前的問題：勾選現場條件後，左側場景示意圖都長一樣＝沒有連動。請務必修正，讓圖示**每次選擇/勾選都即時改變**（切換用平滑過渡/淡入動畫）：

| 影響來源 | 場景要跟著變 |
|---------|-------------|
| 場地大小 | 場地框比例變大/變小 |
| 人數 | 框內人形數量（取人數代表值，靜態不走動） |
| 勾「室外/強光」 | 戶外背景（天空/草地）、明亮、有窗/陽光感 |
| 勾「照明不足」 | 偏暗場景 |
| 勾「可鋪地墊」 | 地面顯示地墊/燈條 |
| 勾「可架上方桁架」 | 顯示上方桁架 + 相機 |
| 未勾「可架桁架」但用 OptiTrack（大型） | 顯示**落地支架**相機（不是吊掛桁架） |
| 勾「需移動/非固定場館」 | 簡易可攜設備、少基礎建設 |
| 最終定位方案 | 對應設備（地墊/桁架/落地支架/角落基站/免施工不畫） |

> 重點：**圖示不可一成不變**；任一選項變動，畫面立即重繪。

# 2. 右上角三語切換

- 右上角放**語言切換**：繁體中文（預設）/ English / 日本語。
- 切換後**全站文字即時換語言**（標題、情境、武器、場地、現場條件、方案內容、價格標籤、按鈕…）。
- **產品專有名詞維持原文**（Antilatency、OptiTrack、Focus Vision、Vive、Sensoryx…）。
- 文字集中成一份 i18n 對照（三語 key-value），方便維護；主要字串對照見附錄，其餘請一致翻譯。

# 3. OptiTrack 落地支架規則

- **大型場地 + 未勾「可架上方桁架」** → OptiTrack 改用**落地支架**：仍可選、**成本較低**、場景畫落地支架相機。
- **超大型場地** → OptiTrack **只能上方桁架**（落地支架不適用）；若超大型又未勾「可架桁架」 → **OptiTrack 排除**。
- 勾「可架上方桁架」→ OptiTrack 用上方桁架（一般）。

# 4. 現場條件（精簡為 6 項）

☐ 可架上方桁架　☐ 可鋪地墊　☐ 室外/強光　☐ 照明不足　☐ 需移動(非固定場館)　☐ 供應鏈限制(避中國大陸)
- 勾選＝現場具備此條件（未勾＝不可/不具備）。**勾選要真正連動**場景圖示與定位篩選（至少先做到圖示連動 + OptiTrack 落地支架規則）。
- 移除原「需快速佈置(1hr)」。

# 其餘規格（沿用 v7，不變）

- **武器數量可調 + 量大折扣**（第1把原價→2–5把7折→6–15把5折→16+把3折）；預設 手槍/步槍/警棍/電擊槍/辣椒水罐＝人數，其他＝1。**價格本版不動**。
- **定位推薦 + 備選**：篩出符合者，標「推薦」（成本最低）+ 其他可點選切換；WT 只用 1～4 人。
- 藍 UI `#0071e3` + 紅 logo；頂部全寬價格列凍結、**價格字級小、手機一行**。
- 左欄（武器轉盤＋場景示意圖）凍結；右欄選單捲動。右欄順序：快速套用→情境→武器(多選+數量)→場地→人數→現場條件→定位結果(推薦+備選)。
- 場地↔人數上限：小型≤5–8、中型≤9–16、大型/超大不限；滅火器限1–4。
- 武器轉盤：去背照(`uploads/槍枝照片/`)、無限循環、限框內、中央130%、間距+30%、新增跳最前。
- 方案內容(BOM)：`uploads/pic/` 真實設備照（含 `Wrist Tracker.png`）、白底圓角卡、不強調廠牌、數量反映武器數量。
- 快速套用：滾輪可左右捲動 + 按鈕按壓特效。所有可互動元素 hover 上浮 highlight。
- Apple 大圓角；字體 `-apple-system,"SF Pro Display","SF Pro Text","PingFang TC","Noto Sans TC"`；情境上方不放藍點。
- 圖片路徑指向 `uploads/`，不要自編 `assets/w-*.png`。純前端單檔、資料集中、程式碼乾淨有註解。
