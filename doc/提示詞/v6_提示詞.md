# Claude Design 提示詞 v6 — 模擬訓練解決方案配置器

> 依「產品配置器6」會議的**策略性轉變**：不再讓使用者選定位方式，改為勾「現場條件」→ 自動給單一最優解。
> 使用方式：在現有 v5 成品上續做——開頭加「請以我現有的版本為基礎，套用以下優化」，
> 再貼「===提示詞開始===」到「===提示詞結束===」+ 附錄；確認 `槍枝照片/`、`pic/` 已上傳。

## 本版改動摘要（相對 v5）

1. **移除「追蹤定位方式」選擇器**：使用者不再直接選定位系統（業務/客戶不懂技術，也不該懂）。
2. **新增「現場條件」勾選區**（取代定位選擇）：以現場實況為篩選依據。
3. **自動最優解**：依「場地 + 人數 + 現場條件」走篩選 → 若多個定位皆符合，**自動選成本最低者**。
4. **前臺不顯示定位廠牌**；最終用到的設備只在**「方案內容」**呈現（圖片 + 泛稱，不強調品牌）。
5. **勾選現場條件時跳出對應示意圖**（如勾「可鋪地墊」顯示地墊示意圖）。
6. **價格維持顯示**（是否隱藏屬銷售策略，保留日後可關閉的彈性）。
7. **數量**本版先不做（之後再加）。
8. **快速套用課程：滾輪可左右捲動**（再次確認要生效）。
9. **規格摘要用白話**、更貼近使用者。
10. 其餘 v5 規格沿用（藍 UI+紅 logo、左視覺凍結右選單捲動、價格頂部全寬凍結、武器轉盤、hover 上浮、Apple 圓角+SF Pro、場地↔人數上限、滅火器限1–4…）。

# 角色

你是資深 **產品設計師 + 前端工程師**，擅長 Apple 官網等級的極簡乾淨互動網頁。
請在我現有版本基礎上，做這個**模擬訓練解決方案配置器**（純前端、可直接開啟）。

# 本版核心轉變：定位方式改為「現場條件 → 自動最優解」

**重要**：移除原本讓使用者「選定位方式（Antilatency/OptiTrack…）」的選擇器。
業務與客戶不懂也不需要懂定位技術或廠牌。改為：

### 右欄新增「現場條件」勾選區（取代原定位選擇器）
以白話、貼近現場的條件供勾選（可複選）：
- ☐ 場地可架設桁架（Truss）
- ☐ 場地可鋪設地墊
- ☐ 室外或強光環境（窗戶多、日照強）
- ☐ 照明不足環境
- ☐ 需移動使用（非固定場館）
- ☐ 需快速佈置／收拾（1 小時內完成）
- ☐ 供應鏈限制（避免特定產地，如中國大陸）

**勾選回饋**：勾某些條件時跳出對應**示意圖**（例：勾「可鋪設地墊」→ 顯示地墊鋪設示意圖）。

### 自動最優解（背後引擎）
- 依「場地大小 + 人數 + 已勾現場條件」篩選出**相容的定位方案**。
- 若多個皆符合，**自動選成本最低者**。
- 使用者**看不到也不用選**定位方式；**前臺不顯示定位廠牌名稱**。
- 最終選到的設備，只在下方**「方案內容」**呈現（設備圖 + 泛稱，如「定位設備」），不強調品牌。

### 現場條件 → 篩選邏輯（合理預設，技術待校正）
| 勾選條件 | 影響 |
|---------|------|
| 不可架桁架 | 排除需桁架的方案（OptiTrack） |
| 不可鋪地墊 | 排除需鋪地墊的方案（Antilatency 地墊式） |
| 室外/強光 | 排除受強光影響的光學方案（OptiTrack 等）→ 傾向免施工/抗光方案 |
| 照明不足 | 傾向不依賴環境光的方案 |
| 需移動/快速佈置 | 排除需大量施工者（OptiTrack/Antilatency/Vive 基站）→ 傾向免施工（Ultimate Tracker / Wrist Tracker / Sensoryx） |
| 供應鏈限制 | 排除受限產地之廠牌（清單待技術單位提供） |

> 篩選後若無任何方案符合，顯示「需技術單位評估」提示，不報錯。

# 其餘規格（沿用 v5）

- **價格維持顯示**於頂部全寬凍結列（保留日後可隱藏的開關彈性）；UI 藍 #0071e3 + 紅 logo。
- 版面：頂部價格列凍結；左欄（武器轉盤＋場景示意圖）凍結永遠在；右欄選單捲動。第一屏看到武器＋場地＋現場條件。
- 右欄順序：快速套用 → 應用情境（警勤/軍事/消防，選中藍、不換色）→ 武器與裝備（多選，重點）→ 場地大小 → 人數 →（新）現場條件。
- **場地↔人數上限**：小型≤5–8、中型≤9–16、大型/超大型不限；切換場地超上限自動下修。
- **滅火器**：人數限 1–4。
- 武器轉盤：去背產品照、無限循環、限框內不溢出、中央放大130%、間距+30%、新增跳最前；圖用 `uploads/槍枝照片/`。
- **方案內容(BOM)**：用 `uploads/pic/` 真實設備照（頭盔 + 自動選到的定位設備），統一白底圓角卡片；**不強調廠牌**。
- **快速套用課程：滑鼠滾輪可左右橫向捲動** + 按鈕按壓特效。
- **所有可互動元素 hover 都要上浮 + highlight**（最高優先）。
- 場景示意圖：場地框＋人形（人數代表值、靜態）＋依自動選到的定位設備畫設備（地墊/桁架/基站/免施工不畫）。
- Apple 連續曲率大圓角；字體 `-apple-system, "SF Pro Display", "SF Pro Text", "PingFang TC", "Noto Sans TC"`。
- **數量本版先不做**（保留之後加）。
- 規格摘要用白話、貼近使用者。
- 圖片路徑指向 `uploads/`，**不要自編 `assets/w-*.png` 佔位**。
- 桌機優先；純前端單檔、資料集中於一物件、程式碼乾淨有註解。
