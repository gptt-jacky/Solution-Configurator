## v5 → v6 主要改動（★策略轉變，依「產品配置器6」會議）

1. **移除定位方式選擇器**：使用者不選技術/廠牌
2. **新增「現場條件」勾選**（可架桁架/可鋪地墊/室外強光/照明不足/需移動/快速佈置/供應鏈限制）
3. 引擎依 場地+人數+現場條件 → **自動取成本最低的定位方案**
4. 前臺不顯示廠牌；最終設備只在「方案內容」呈現
5. 勾條件跳示意圖；價格維持顯示（可日後隱藏）
6. 數量本版先不做；快速套用滾輪可左右捲動

## v4 → v5 主要改動

1. **場地限制人數上限**：小型≤5–8、中型≤9–16、大型/超大型不限
2. **Antilatency 不支援 25–40 人**（人數 25–40 時自動禁用）
3. 切換場地時若人數超上限自動下修
4. **圖片路徑接對 `uploads/`**（武器→`uploads/槍枝照片/`、BOM→`uploads/pic/`），不要編 `assets/w-*.png` 佔位

## 使用方式

1. Claude Design 新專案選 **Prototype**，Model 用 Claude Opus 4.8。
2. 複製該版「===提示詞開始===」到「===提示詞結束===」整段 + 下方「內容資料附錄」。
3. **上傳 `槍枝照片/` 資料夾**（武器轉盤用，檔名對應見附錄）。
4. **貼上橫向 Logo 圖**（全球動力科技，緋紅 ≈ #D6173E；放頂部）。
5. （可選）附 OptiTrack「Build My System」截圖當 UX 參考。
