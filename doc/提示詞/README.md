## v2 → v3 主要改動（依「產品配置器4」會議）

1. **顏色全站統一藍**（取消情境換色）
2. **價格置頂凍結**（頂部全寬列，永遠藍）
3. **凍結方向反轉**：左視覺凍結、右選單捲動
4. 封面一屏看完，**武器在場地之上**
5. R 角再加大（更明顯 squircle）
6. 字體比照 Apple（SF Pro / 蘋方）
7. 武器轉盤無限循環、限制在框內不溢出
8. 滅火器人數限 1–4
9. 桌機優先，手機之後

## v1 → v2 主要改動

1. 武器改成**轉盤 Carousel**（去背產品照，正中清晰兩側淡化）＝畫面主角
2. 新增**消防**情境（三情境配色：警勤藍/軍事綠/消防橘紅）
3. **hover 上浮 highlight**（硬需求，v1 漏做）
4. 圓角改 **Apple 連續曲率 squircle**
5. 人數改**範圍級距** 1 / 2–4 / 5–8 / 9–16 / 17–24 / 25–40
6. **右欄凍結、左欄捲動**，第一頁看到全部重點
7. 場景圖加大、定位設備分方式畫（AL 地板燈條…）
8. BOM 卡片放大、用自家產品圖（不外連他牌）
9. 定位方式維持低調、不做轉盤

## 使用方式

1. Claude Design 新專案選 **Prototype**，Model 用 Claude Opus 4.8。
2. 複製該版「===提示詞開始===」到「===提示詞結束===」整段 + 下方「內容資料附錄」。
3. **上傳 `槍枝照片/` 資料夾**（v2 武器轉盤會用到，檔名對應見 v2 附錄）。
4. （可選）附 OptiTrack「Build My System」截圖當 UX 參考。
