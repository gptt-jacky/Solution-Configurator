# Claude Design 提示詞 v9 — 模擬訓練解決方案配置器（正式部署版）

> 沿用 v8，本版目標：**正式對外部署**。移除一切內部/開發感/客戶不該看到的內容；補完整武器圖。
> 使用方式：在現有 v8 成品上續做——開頭加「請以我現有的版本為基礎，套用以下優化」，
> 再貼「===提示詞開始===」到「===提示詞結束===」+ 附錄；確認 `槍枝照片/`、`pic/` 已上傳、並貼上場景示意圖。

## 本版改動摘要（相對 v8）

1. **定位結果只顯示「推薦一個」**：移除備選方案、移除「點其他卡片可切換回推薦」等說明文字與切換 UI。引擎仍自動選成本最低，但**畫面只呈現那一個**。
2. **價格只顯示乾淨總價**：**不顯示**量大折扣機制（7折/5折/3折）、開發費、成本拆解、明細。折扣仍反映在總價數字裡，但**不揭露機制**。
3. **移除所有內部/開發字樣**：不得出現「預估/假價/待校正/廠牌名/debug/佔位/備選/切換」等；全站文案**專業、對外**。
4. **補完整武器圖**：榴彈槍/警棍/辣椒水罐/無人機/悍馬車/快艇/滅火器 都接上去背圖（檔名見附錄）。
5. 其餘 v8 規格全部沿用（三語切換、場景示意圖隨選擇連動、現場條件篩選、武器數量、場地↔人數上限、藍UI+紅logo、左視覺凍結右選單捲動、價格頂部凍結且手機一行、Apple圓角+SF Pro、hover上浮…）。

# 角色

你是資深 **產品設計師 + 前端工程師**，擅長 Apple 官網等級的極簡乾淨互動網頁。
這是一個即將**正式對外部署**的產品配置器，請以**上線品質**完成：文案專業、畫面乾淨、無任何內部/開發痕跡。

# 產品背景

軍警消 XR 模擬訓練公司（全球動力科技 / BigxReality）的**模擬訓練解決方案配置器**，給業務面對客戶用。
使用者勾選需求 → 自動配出方案、設備與報價。

# 正式部署：對外清理（最重要）

**畫面上絕對不可出現以下「內部/開發感」內容：**
- ❌ 「備選方案」「點其他卡片可切換回推薦」之類**操作說明/開發微文案**。
- ❌ **量大折扣、7折/5折/3折、開發費、製作費、成本拆解、明細**等定價機制字眼。
- ❌ 「預估」「假價」「待校正」「廠牌名」「placeholder」「示意」等字樣。
- ❌ **說明性/內部措辭**：如「隨人數」「預設數量」「每人一把」「依場地自動篩選」等。數量與結果**直接呈現數字/方案**即可，不要在畫面解釋背後邏輯。
- ❌ 任何 debug、TODO、佔位文字。

**正面要求：**
- **定位結果只呈現「一個方案」**（引擎自動選成本最低的相容方案），直接顯示「為你配置的系統」與其設備；不顯示其他選項、不提供切換。
- **價格只顯示一個乾淨總價**（專業措辭，如「方案總價」），折扣已反映在數字內但不揭露。
- 全站文案專業、簡潔、對外（繁中/英/日三語皆是）。

# 版面（沿用 v8）

- 頂部全寬列（凍結）：左＝橫向 Logo（全球動力科技）；右＝**語言切換（繁中/English/日本語）** + **方案總價（字級小、手機一行）** + 主要 CTA（如「匯出報價」）。
- 左欄（凍結，永遠在）：**武器轉盤** + **場景示意圖**（隨選擇即時變化）。
- 右欄（捲動）：快速套用 → 應用情境 → 武器與裝備（多選＋數量）→ 場地大小 → 人數 → 現場條件。
- 下方：方案內容（設備清單，真實設備圖）+ 規格摘要。

# 場景示意圖（沿用 v8，務必真連動）

- 用使用者在對話貼上的場景示意圖，依**最終定位方案**對應切換（落地支架/上方桁架/地墊/角落基站/免布置）。
- 並隨場地大小（框比例）、人數（人形數）、現場條件（室外/強光/照明不足/地墊/桁架/需移動）即時變化，切換用平滑動畫。
- 圖示不可一成不變。

# 定位引擎（背後邏輯，畫面只出結果）

依「場地 + 人數 + 現場條件」篩出相容方案，**自動取成本最低者為最終方案**（畫面只呈現它）：
- 場地↔人數上限：小型≤5–8、中型≤9–16、大型/超大型不限。
- 適用場地：Vive(小)、UT/WT/SX(小/中)、AL(小/中/大/超大)、OptiTrack(大/超大)。
- WT 只用 1–4 人。
- 未具「可鋪地墊」→ 排除 AL；未具「可架上方桁架」→ 大型 OptiTrack 改落地支架(較便宜)、超大型排除 OptiTrack。
- 勾「需移動/非固定」→ 排除需施工(AL/OptiTrack/Vive)。
- 勾「照明不足」→ 排除 Ultimate Tracker（需光線充足）。
- 若無相容方案 → 顯示專業提示（如「請與我們聯繫以取得客製方案」），不報錯、不顯示技術細節。
