## 1. 決策樹（四步）

```
① 選場地 ──→ 限定人數上限 + 初步候選方案
   小型(<36㎡)   人數≤5–8   候選：Vive / UT / WT / SX / AL
   中型(36–81)   人數≤9–16  候選：UT / WT / SX / AL
   大型(81–300)  人數不限   候選：AL / OptiTrack(桁架或落地)
   超大型(>300)  人數不限   候選：AL / OptiTrack(僅桁架)

② 選人數 ──→ 進一步排除
   WT 只留 1–4 人（≥5 人排除 WT）
   25–40 人 → 排除 AL [待校正]（會議曾說純人可到40，待技術定）

③ 勾現場條件 ──→ 逐條篩
   未勾「可鋪地墊」     → 排除 Antilatency
   未勾「可架上方桁架」 → 大型：OptiTrack 改落地支架(較便宜)；超大型：排除 OptiTrack
   勾「需移動/非固定」  → 排除需施工者(AL / OptiTrack / Vive) → 只剩免施工(UT/WT/SX)
   勾「室外/強光」      → [待校正]（暫定排除 OptiTrack 等光學）
   勾「照明不足」        → 排除 Ultimate Tracker（UT 需光線充足）
   勾「供應鏈限制」      → [待校正]（受限廠牌清單未定）

④ 剩下的方案 ──→ 成本最低＝【推薦】，其餘依成本由低到高列為【備選】
   （成本 = 每人單價 × 人數 + 基礎建置）
```
