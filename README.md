# 解決方案配置器 (Solution Configurator)

> BigX Reality（大鵬實境）公司的**模擬訓練解決方案配置器**。
> 給**業務／行銷**用的工具：協助面對政府／軍方客戶時，把客戶模糊的需求
> （場地多大、要練什麼武器、幾人）轉成**具體方案 + 價格 + 視覺示意圖**。
> 計算邏輯來自內部「XR解決方案成本試算」Excel。

---

## 專案定位

- **使用者**：業務／行銷（非終端消費者；面對政府/軍方客戶）
- **核心痛點**：新客戶不知道自己要什麼，業務以前接不起來前期諮詢
- **目標**：客戶勾選需求 → 引導式流程 → **自動排除不相容選項** → 輸出推薦方案 + 價格 + 示意圖
- **互動體驗**：**汽車配置器式**——選不同組合時，左側場域模擬示意圖即時跟著變化
  （標竿：OptiTrack「Build My System」，見 [doc/參考網站.md](doc/參考網站.md)）
- **技術形式**：純前端網頁（HTML/CSS/JS 或 React；資料以 JSON 驅動，可直接部署）
- **MVP 態度**：先做骨架 + 假資料，價格先粗估，再逐步填真資料
- **設計風格**：之後以 Claude Design 產生，對齊公司官網 <https://www.bigxreality.com/tw/home> 的科技感／極簡專業調性

> 需求細節見 [doc/需求_會議紀要.md](doc/需求_會議紀要.md)。

## 資料來源

- `XR解決方案成本試算.xlsx`：46 張工作表
  - **主表**「定位方式場域人數成本總表」：6 種定位方式 × 5 種追蹤配置的成本矩陣（核心邏輯）
  - 其餘 45 張：各客戶實際案例（軍警消、海巡、海外案場等）與產品線價格表
- 資料模型整理見 [DATA_MODEL.md](DATA_MODEL.md)
